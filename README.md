# EQUIPE_2

Conceito:

Desenvolver uma solução para prever Desastres Naturais ligados ao clima. Foco em empresas ou sociedade

Nossa solução será voltada para empresa (seguradora)

## Solução:


Plataforma digital de informação, voltada a seguradoras, com o objetivo de demonstrar a média percentual de risco de desastres naturais (enchentes), podendo escolher a estação do ano e a região da cidade de São Paulo desejada. Ao abrir a aplicação, o usuário (seguradora) se conectará automaticamente, por meio da localização do dispositivo, a uma API que o localizará no mapa e permitirá buscas de locais no mapa. Como entrada de dados, haverá a rua do contratante da seguradora, na qual a API encontrará o endereço. Sabendo este endereço, uma IA irá realizar uma varredura na web e nas redes sociais com o objetivo de captar notícias e comentários de pessoas sobre enchentes na localidade selecionada, bem como naquela estação do ano (utilizará dados oficiais e de postagens e sites na internet). A IA, de posse destas informações, realizará o cálculo do percentual de risco de enchente na área, separando uma categorias (baixo, médio e alto) e retornará, na tela, ao usuário este valor e esta categoria. Haverá, na tela também, local para editar a conta, notícias sobre a cidade de São Paulo, e uma aba dedicada a informar boas práticas para que o cidadão pode realizar para mitigar a chance de novas enchentes. API Weather do Google Maps pode ser usada para fornecer dados climáticos em tempo real, ajudando no funcionamento da IA preditiva. Google Flood Hub para fazer a predição de novas enchentes em SP.

A PLATAFORMA FICA RESTRITA APENAS A CIDADE DE SÃO PAULO.


## Como o projeto poderá gerar lucro?

Serão postos na aplicação anúncios via AdSense (Google) na versão gratuita, enquanto na versão paga não haverá qualquer tipo de anúncio, sendo a versão ideal para empresas e corporações.

## Como divulgar o projeto?

Criar um site de publicidade para divulgação, bem como perfil em mídias sociais, e entrar em contato direto com empresas (contato comercial) oferecendo o serviço da aplicação.

API utilizada: Nominatim, API Weather do Google Maps e Google Flood Hub

IA utilizada: Gemini
